### Termux-x11 Kali-Nethunter
Commands in Termux
> pkg install nano -y

* Edit script
```
nano ../usr/bin/nethunter
```

* Add script
```
-b /data/data/com.termux/files/usr/tmp:/tmp \
```
Save : ctrl + x, click Y enter.
